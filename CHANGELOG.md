### Added
- Initial CLI implementation with interactive task management
- `configure` command for API token setup
- `tasks select` interactive task browser
- `tasks list` for recent assigned tasks
- `tasks due` for tasks due soon (reminders)
- `tasks get` for task details
- `tasks move` for status changes
- Add comment functionality
- Open in browser functionality
